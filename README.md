# ASP.NET-MVC
#### BookMaintain 簡易的圖書館系統
* 使用ASP.NET MVC Framework
* 新增、修改、查詢、刪除書籍
* 可依據書名、圖書類別、借閱人、借閱狀態做查詢
* 已借出的書籍不得進行刪除
